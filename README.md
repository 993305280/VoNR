# CDR管理平台

基于 Vue3 + Vite + Element Plus + ECharts 的后台管理系统
花了几天利用claude code + mimo模型，按照还可以打开的ui设计图模仿出原本需要几个月完成的公司项目。
功能完成度90%，ui还原度80%，换成Gemini模型ui还原度还能更高，期间没有手敲或手修改过一行代码。

## 技术栈

- Vue 3
- Vite
- Element Plus
- ECharts
- Pinia
- Vue Router

## 安装依赖

```bash
npm install
```

## 开发

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 预览

```bash
npm run preview
```
